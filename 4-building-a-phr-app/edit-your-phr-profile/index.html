Edit your profile

User can view and edit his/her profile in the View complete Profile on dashboard. The screen displays QR code which user can use to share the information with HIP and can also download the ABHA card. The screen also displays user details that are KYC verified (Aadhar verified profile) and cannot be edited such as

  1. Name
  2. Gender
  3. DOB Other details displayed are editable such as
  4. Mobile
  5. Email ID
  6. Address
  • District
  • State
  • Pin Code

In self-verified profile (Created by Mobile/Email ID), all the details can be edited.

Swagger link for API: https://app.swaggerhub.com/apis-docs/abdm.abha/abha-service/1.0#/Integrated%20Programs/updateAccountInformationUsingPOST_1
